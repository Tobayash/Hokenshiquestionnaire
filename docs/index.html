保健師の方々を対象とした
コロナ禍のメディア報道と働き方に関する調査

本調査は、福島県立医科大学の小林智之と医療創生大学の吉田和樹が行うものです。
新型コロナウイルスの感染拡大により、健康問題の他にも、働き方やメディア報道の在り方といった様々なことが問題となっております。本調査は、メディア報道が保健医療従事者に注目する中、日々ご活躍の保健師の皆様が業務に対してどのような気持ちや印象を抱いているのかについてお尋ねします。
全国の保健師の方々を対象に、回答者の方々に本アンケートをご紹介いただきながら配布されています。そのため、同じ方に複数回届く可能性がございますこと、あらかじめお詫び申し上げます。
本調査の結果は、保健師の方々にとっての安全で充実した職場を考える大切な基礎資料とします。
ご協力の程よろしくお願いいたします。

以降で、業務の満足度やストレス、みなさまの持っているイメージについてお伺いします。また、メディアの報道記事を提示して、それに対するお気持ちをお伺いすることもいたします。回答時間はおよそ１０分です。
